.outlined

Для input и textarea поддерживается дополнительный визуальный режим отображения .outlined — с обводкой. Используйте этот режим исходя из эстетических соображений и для соответствия внешнего элемента интерфейсу вашего приложения.

В рамках одной формы ввода данных рекомендуется использовать только один из вариантов отображений: либо обычный, либо .outlined. Не смешивайте их друг с другом.
