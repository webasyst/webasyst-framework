Индивидуальные иконки span.icon

Компонент .icon предназначен для ситуаций, когда необходимо вставить свою индивидуальную иконку (например, в <svg> формате или <img>). Помещенная внутрь контейнера span.icon, иконка будет стилистически оформлена в соответствие с текущий кеглем.

Этот класс — необязательный, т. е. все подряд иконки не нужно оборачивать в .icon. Так, иконки Font Awesome в span.icon оборачивать не нужно в связи с разными свойствами геометрии иконок (Font Awesome имеет максимальную ширину 1.25rem, а индивидуальная иконка .icon всегда считается квадратной с размерами 1x1rem).

Типовые ситуации применения .icon:

  • svg без дополнительных стилей может сломать вёрстку, при использовании с .icon она будет находиться внутри контейнера с нормальными размерами.

    <span class="icon"><svg></span>


  • img больших размеров внутри .icon будет сужен до размера контейнера.

    <span class="icon"><img></span>


  • Прочие ситуации, когда нужно иконку, картинку или цвет реализовать через фон. Стили .icon адаптированы для этого.

    Класс .rounded на внутренний элемент сделает иконку круглой. Класс .bordered добавит явную границу (часто может быть полезно в комбинации с .rounded).

    <span class="icon"><i style="background-image: url('{$wa_url}wa-apps/ui/img/ui.svg?v={$wa->version()}');"></i></span>

    <span class="icon"><i class="rounded" style="background: orangered;"></i></span>

    <span class="icon"><i class="rounded bordered" style="background: gold;"></i></span>

    Класс .userpic на внутренний элемент сделает иконку чуть более различимой (1.25rem×1.25rem ≈ 20×20px) и правильно расставит отступы.

    <span class="icon"><i class="userpic" style="background-image: url('/wa-content/img/userpic20@2x.jpg');"></i></span>

Изначально .icon — это квадратный контейнер с размерами 1x1rem. Размерами контейнера можно управлять, используя дополнительный класс .size-xx, где xx — число, обозначающее размер в пикселях при условии 1rem = 16px (при изменении размера rem иконки масштабируются аналогичным образом).

span.icon.baseline

Для управления вертикальной позицией элементов относительно друг друга используйте дополнительные классы:

  • .icon.top — верх строки
  • .icon.baseline — базовая линия
  • .icon.middle — середина строки
  • .icon.shift-x — со смещением вниз на x пикселей относительно baseline. Актуально в ситуациях с разными размерами шрифта у соседних элементов.
