Компонент общего назначения .inlinebox предусматривает вывод элементов в виде inline-block списка. Рекомендуется к использованию в простых ситуациях, когда нет нужды реализовывать сложные конструкции через .flexbox или .tablebox.

Все элементы выстраиваются в горизонтальный ряд, прижимаясь по вертикали к верхней границе блока (vertical-align: top).

Для управления вертикальной позицией элементов относительно друг друга используйте доп. классы: .inlinebox.baseline .inlinebox.middle

Никакого специфического для бекенда Webasyst оформления этот класс не навязывает, но будет полезен в качестве структурного контейнера, содержащего разные интерфейсные компоненты внутри: заголовки, меню, кнопки, инпуты, блоки с произвольным контентом .box.

.inlinebox.space-16

С помощью дополнительных классов .inlinebox.space-XX добавляются внутренние отступы для дочерних элементов, где XX — значение в пикселях с шагом 4 — 0, 4, 8, 12, 16, 20, 24, 28, 32, 36, 40, 44, 48. Шаг в 4 пикселя выбран исходя из соображений сохранения визуальной сетки интерфейса и практической полезности. Пиксели будут пересчитаны в rem.
