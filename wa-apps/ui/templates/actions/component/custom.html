Отступы и интервалы — .custom-*

Форсированная корректировка отступов и отбивок (padding и margin) для всевозможных элементов. Используйте, только если стандартная компоновка никак не позволяет достичь нужного результата, и вы точно знаете, что делаете.

Синтаксис довольно простой: .custom-WHAT-VALUE или .custom-WHAT-VALUE-mobile.

WHAT:

  • m = margin
  • mt = margin-top
  • mb = margin-bottom
  • ml = margin-left
  • mr = margin-right
  • mx = margin-left && margin-right
  • my = margin-top && margin-bottom

  • p = padding
  • pt = padding-top
  • pb = padding-bottom
  • pl = padding-left
  • pr = padding-right
  • px = padding-left && padding-right
  • py = padding-top && padding-bottom

VALUE — значение в пикселях — 0, 2, 4, 8, 12, 16, 20, 24, 28, 32, 36, 40, 44, 48. Шаг в 4 пикселя выбран исходя из соображений сохранения визуальной сетки интерфейса и практической полезности. Пиксели будут пересчитаны в rem.

Суффикс -mobile позволяет применить правило только на мобильных устройствах (ширина экрана <= 760px).

Например:

{_render_1|escape}

Применять можно к любому элементу (компоненту) интерфейса, так как все эти классы используют !important.

Классы можно совмещать друг с другом.
